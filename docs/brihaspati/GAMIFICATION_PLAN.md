# Gamification & Social Identity Plan

## The Cosmic Identity Card

### Identity Dimensions (6)
Every user has a unique cosmic fingerprint composed of:

1. **Lagna Archetype** (1 of 12) — core self, visual artwork
2. **Moon Rashi** (1-12) — emotional identity, green band
3. **Nakshatra** (1-27) — stellar signature, purple band
4. **Birth Tithi** (1-30) — lunar day energy, pink band
5. **Birth Masa** (1-12) — Vedic month, part of the cosmic number
6. **Mahadasha Lord** (1-9) — current life phase, blue band
7. **Transit Tribe** — shared planetary weather, amber band

### The Cosmic Number
Displayed prominently: `7 · 15 · 1 · 10 · 5`
Format: `Rashi · Nakshatra · Masa · Tithi · Graha`

This is the user's celestial coordinates. Two people with identical
numbers were born under the same lunar conditions.

### Visual Design
- Full SVG artwork per archetype (12 unique illustrations)
- Art style: refined, elegant, inspired by classical Indian iconography
  — NOT cartoon/caricature. Think temple sculpture meets modern design.
- Text overlay on dark gradient for readability
- Frosted glass data grid for the 6 dimensions
- Each archetype has its own colour palette

### Sharing
- Instagram story (9:16) and WhatsApp (1:1) variants
- QR code watermark linking to dekhopanchang.com
- "Compare with a friend" feature via shareable link

### Implementation
- Auto-generated for all users with birth data
- Displayed on dashboard as the identity hero card
- Computed from kundali snapshot data (already available)
