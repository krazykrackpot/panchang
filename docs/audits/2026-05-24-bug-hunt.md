## Cross-cutting themes

1. **The Razorpay/INR payment surface is the missing twin of the Stripe hardening.** Sprint 1–7 fixed Stripe end-to-end (event-id idempotency, server-side `pending_checkouts` binding, fail-loud DB writes). The Razorpay branch — both the main subscription webhook and the Brihaspati twin — never received any of those fixes. Three separate audit sections (silent-failures SF-1/2/3, security SEC-1/2/3/4, idempotency IDEM-1/2) report the same cluster. **This is the highest-blast-radius unresolved class.**

2. **The `getSunTimes` tz-safe contract migration is half-done.** Sprint 4 added `*Minutes` numeric fields to the return type but left the deprecated `Date` fields in. Six consumers still call `.getHours()/.getMinutes()` on those Dates: `shadbala.ts`, `puja/muhurta-compute.ts`, `parana-compute.ts`, `gochar.ts`, `transit-alerts.ts`, `journal/snapshot.ts`. Until the Date fields are *removed* from `SunTimes`, the compiler can't enforce the migration.

3. **Cron dedup anchors are still fail-open across at least four jobs.** `generate-notifications`, `transit-alerts`, `email-alerts`, `monthly-readings` build their "did we already do this" sets from queries that swallow errors. One DB blip → empty set → every alert re-fires. This is structurally the May-20 welcome-email bombardment shape.

4. **AI quota gates are TOCTOU-racy (money + brand)**. Three AI endpoints (`ai-reading`, `domain-pandit`, `tippanni-llm`) and the central `checkAndIncrementUsage` use read-then-increment with the LLM call in the window. Parallel requests bypass plan limits; across Fluid Compute containers the in-memory `Map` doesn't share, so the bypass is `N × limit`.

5. **Lesson Q half-applied.** Five constant clusters still hold local copies — `FRIENDS/ENEMIES` (5 files; 2 actively divergent for Rahu/Ketu), `NAKSHATRA_LORDS` (5 files), `COMBUSTION_ORBS` (3 files), `SIGN_LORD` (3 files), `CHALDEAN` (2 starting points). One concentrated PR could close ~15 callers.

6. **Non-EN locales are still leaky in UX-critical chrome.** AuthModal is entirely English across 10 locales; `not-found.tsx` and `error.tsx` are English-only; pricing checkout error toast is English-only; profile page contains a Tamil string inside the `en` block.
