Design Notes

Variant 1 (Card) is better for: sharing, comparison, readability, WhatsApp/Instagram stories. Each band is immediately scannable. The "✦ MATCH" highlight makes comparison instant.

Variant 2 (Ring) is better for: the profile page, the "cosmic fingerprint" feel, visual wow factor. The slowly rotating rings feel alive. But harder to read at a glance and doesn't compare well.

Recommendation: Use the Card for sharing + comparison. Use the Ring as the profile page hero (animated, interactive — hover each ring to see details). Both generated from the same data.

Colour system (6 identity dimensions):

• Gold (#d4a853) = Archetype — the core self (lagna-based)

• Green (#22c55e) = Moon/Rashi — emotional identity (number = rashi 1-12)

• Purple (#8b5cf6) = Nakshatra — stellar signature (number = nakshatra 1-27)

• Pink (#ec4899) = Birth Panchang — tithi + masa + paksha (number = tithi 1-30)

• Blue (#3b82f6) = Dasha — temporal identity (number = planet id)

• Amber (#f59e0b) = Transit — current cosmic weather

Numerology numbers: Each band shows a circled number — rashi number (1-12), nakshatra number (1-27), tithi number (1-30), planet number for dasha lord. These are the "coordinates" of your cosmic identity. Numerology enthusiasts can instantly compare these numbers across cards.

Birth Panchang band: Shows the tithi + paksha + masa at birth. This is deeply personal — the tithi you were born on has specific qualities in Jyotish (e.g., born on Panchami = Lakshmi tithi, born on Ashtami = challenging but transformative). The Vikram Samvat year adds another identity layer.

Comparison mechanic: When two cards are side by side, matching bands get a gold border + "✦ MATCH" label. The tribe strip at the bottom shows "3 of 5 bands match — cosmic kindred" or "1 of 5 — complementary energies" depending on overlap count. Same rashi + same dasha = deeply similar life phase. Same tithi = born under the same lunar energy.
