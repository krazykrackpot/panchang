Tithi Calendar — Festival Icons (Mockup v1)

Visual review for the festival SVG set that will land in src/components/icons/FestivalIcons.tsx as part of Phase 2. All icons match the existing AshtamangalaIcons.tsx style: 48×48 viewBox, gold gradient (#f0d48a → #d4a853 → #8a6d2b), subtle glow filter, 1.5–2px strokes, translucent fills at 0.1–0.3. Each icon is shown at the 40px detail-panel size (left) and the 16px cell-ribbon size (right). Please review and flag any to redo before I commit them.

This v1 contains ~25 fully-designed icons across all 7 categories. After style approval, the remaining ~50 (mostly Ekadashi-variant overlays and minor festivals) will be finalised using the same vocabulary in Phase 2. Placeholder cards (dashed border) show the slot, the intended motif, and the festival list it covers.
