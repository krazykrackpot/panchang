## Visualization Strategy

Every new page should have:
1. **At least 1 SVG diagram** (interactive where possible)
2. **Data cards** with hover/expand for details
3. **Comparison tables** (side-by-side analysis)
4. **Color-coded classifications** (auspicious=emerald, inauspicious=red, neutral=amber)
5. **Classical verse citations** (BPHS chapter references)
6. **"Try It" links** to relevant tools in the app
